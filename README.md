# cargo-sdl-apk
Build Android packages that uses SDL

This cargo tool aims to be for [Rust SDL](https://docs.rs/sdl2/latest/sdl2/) what
[cargo-apk](https://crates.io/crates/cargo-apk) and [cargo-quad-apk](https://crates.io/crates/cargo-quad-apk) 
is for [Glutin](https://crates.io/crates/glutin) and [Miniquad](https://crates.io/crates/miniquad).
